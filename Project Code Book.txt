Feature Selection 
=================

Only a subset of the original raw data was used for this project. 

The set of variables that were left were those that included the following calculations....

mean(): Mean value  ** Only used for the project **
std(): Standard deviation ** Only used for the project **
meanFreq(): Weighted average of the frequency components to obtain a mean frequency ** Only used for this project **

'-XYZ' is used to denote 3-axial signals in the X, Y and Z directions.
